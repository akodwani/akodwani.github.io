What do the status badges mean?

Spec-mapped means the address matches the documented DJ deck MIDI list, not that it has been tested on real hardware. Physically verified (green) is only set by completing "Verify entire controller" on the real device and confirming each control yourself -- Simulator Mode can never set it. Recognized - pending means a real message is matched but no musical behavior is wired up yet. Unsupported means this app's audio architecture cannot drive that control at all (for example, no microphone input path exists).
